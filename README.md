# xmind-recruitment-project

## 安装依赖
```
yarn install
```

### 运行服务端（无数据库，文件读写模拟数据库操作）
```
yarn server
```

### 本地运行项目
```
yarn serve
```

### 项目打包
```
yarn build
```

### 描述
- 由于项目数据没有导入至数据库，故仅在本地文件做读取操作，以模拟数据库操作
- koa2搭建服务端，提供接口访问
- 前端初始化和ui利用vue-cli + vuetify
